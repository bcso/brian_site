W A N D E R L U S T

A multiplayer first person shooter built at Yale Hack with Unity Engine, Windows Azure and Windows Kinect API within 36 hours with a team of 4 people.

M Y O U R T U A L   R E A L I T Y

A project started at Boilermake '14 featuring the marriage of the Thalmic Labs Myo gesture control armband API and Qualcomm's Vuforia API. Users would be able to control a virtual object anchored in augmented reality via the Myo arm band.

A L A R M S Y S T E M

This is an alarm system created using the Arduino Uno. It implements the basis of how home alarm systems operate - via the use of interrupts and timers. Designed and implemented by 3 team members, this is the cumulative project of SYDE192L - Digital Systems Laboratory.

T E X T   C O N V E R T

An early android application leveraging JavaOCR to process an image taken with the device camera and convert the analog text to digital text.

P E B B L E R U N N E R

This is a pebble application created at the Hack the North Hackathon, featuring a watchface that is able to track the users abulatory activity. There is also a novel accompanying chracter animation. This application was mainly developed using the pebble cloud IDE.

T H E   G E M S T O R E

This is an application built using the Angular.JS framework, it is based on the tutorial provided by CodeSchool.

L E A G U E F E T C H

A python application leveraging the League of Legends API to retrieve player data via endpoints. Users may decide to choose between listing their last 10 game history ID and match length, or scraping user in-game-names. This was created mainly as a proof of concept for retrieving data from the Riot datastore.
